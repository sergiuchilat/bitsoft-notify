## Description

[Nest JS](https://github.com/nestjs/nest) Notify App with [TypeORM](https://typeorm.io/)

## Project Installation

```bash
$ npm install
```

## Running the app

```bash
# development(v1)
$ npm run start

# development(v1)
$ nest start

# watch mode(v1)
$ npm run start:dev

# watch mode(v2)
$ nest start --watch

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
